O que é JavaScript?

JavaScript foi criado em 1995 por Brendan Eich, enquanto trabalhava na Netscape Communications Corporation. A motivação principal era adicionar interatividade às páginas web, permitindo que os navegadores respondessem a ações do usuário sem a necessidade de recarregar a página :contentReference[oaicite:0]{index=0}.

A diferença entre execução no navegador e no servidor está no local onde o código é executado. No navegador (lado do cliente), o JavaScript é executado diretamente no computador do usuário, permitindo interações rápidas e dinâmicas. No servidor (lado do servidor), o JavaScript é executado em servidores web, processando dados e gerando conteúdo antes de enviá-lo ao cliente :contentReference[oaicite:1]{index=1}.

O conceito de "sandbox" no navegador refere-se a um ambiente isolado onde o código JavaScript é executado. Isso significa que o código tem acesso restrito aos recursos do sistema, aumentando a segurança ao evitar que scripts maliciosos acessem ou modifiquem dados sensíveis :contentReference[oaicite:2]{index=2}.

Alguns objetos globais importantes em JavaScript incluem:

  • document: Representa o conteúdo da página web e permite manipulá-lo.
  • console: Usado para exibir informações no console do navegador, útil para depuração.
  • alert(): Exibe uma caixa de diálogo com uma mensagem para o usuário.
